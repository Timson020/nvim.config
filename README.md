## 安装

### MacOS

- Brew(推荐)
```bash
brew install neovim
```

- 安装vim-plug插件管理
```bash
curl -fLo ~/.local/share/nvim/site/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

### Linunx

- Yum
```bash
yum install -y neovim python36-neovim
```

- 安装vim-plug插件管理同Mac OS 方法一致

## 使用

- 启动
```bash
nvim

nvim ./xx
```
